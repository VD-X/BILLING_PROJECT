==============================================
GROCERY BILLING SYSTEM
==============================================

REQUIREMENTS:
- Windows 10 or 11
- Python 3.8 or higher (https://www.python.org/downloads/)
- Internet connection (for first-time setup)

INSTALLATION:
1. Extract this ZIP file to any location on your computer
2. Double-click the "Setup.bat" file
3. Follow the on-screen instructions

USAGE:
1. Launch the application using the desktop shortcut created during setup
2. The application will open in your web browser
3. Follow the on-screen instructions to create and manage bills

TROUBLESHOOTING:
If you encounter any issues:
1. Make sure Python is installed correctly
2. Try running the application directly by opening a command prompt in the 
   application folder and typing: python -m streamlit run Home.py
3. Contact the developer for assistance
